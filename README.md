## Getting Started
### Prerequisites:
To run the api on your local system, you need the following:
* Node.js

### Try local
1. Clone the repo
```sh
git clone https://github.com/24Roger/editor-client.git
```
2. Install NPM packages
```sh
npm install
```
3. Run the following command
```sh
npm run dev
```
